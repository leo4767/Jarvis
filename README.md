Jarvis AI Local

Asistente de inteligencia artificial personal con voz, memoria y control del sistema, funcionando completamente en local.

Funciones

* 🎤 Reconocimiento de voz offline
* 🧠 IA local con Ollama (Llama3)
* 💾 Memoria persistente
* 🖥️ Control del sistema
* 🧑‍💻 Asistente de programación

Instalación

```bash
pip install -r requirements.txt
```

Instalar Ollama:
https://ollama.com

```bash
ollama run llama3
```

 Uso

```bash
python main.py
```

 Ejemplos

* "Hey Jarvis abre chrome"
* "Hey Jarvis terminal dir"
* "Hey Jarvis mi nombre es Leónidas"
